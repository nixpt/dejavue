# Example: seeded-by-chatgpt

The original dejavue origin story — the ChatGPT conversation that conceived the project
(2026-05-13, before the first commit).

This `.dejavue/` directory was captured during that conversation and migrated to the
current layout. It shows what a real dejavue memory looks like from day zero:
a handful of founding decisions, a timeline of the design session, a state snapshot,
and a handoff for the next agent.

## Browse

- `.dejavue/decisions.md` — four founding decisions (use a dir, git is source of truth, JSONL timeline, capture intent not raw thinking)
- `.dejavue/timeline.jsonl` — the 2026-05-13 design session events
- `.dejavue/state.md` — state snapshot from that session
- `.dejavue/handoff.md` — what the next agent (the implementer) needed to know
- `.dejavue/concept.md` — the original concept document written during the conversation
